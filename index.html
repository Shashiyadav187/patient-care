Patient Care Monitoring System

Patient to caretaker IoT web monitoring interface for tracking patient prescription drug usage, providing notes, and pinging notifications.

View on GitHub Download .zip Download .tar.gz

Back-end RESTful API built with Node.js and MongoDB.

Front-end web interface developed with React.

Access deployed site at https://patient-care-web.herokuapp.com

Note: automatic deploys from site-release branch

Access deployed API at https://patient-care-rest.herokuapp.com/api

Note: automatic deploys from api-release branch

Running Patient Care React site locally:

"npm install"
"npm start"
Access at localhost:3000

Integration with Raspberry Pi + DE2 FPGA developed by Alexander Swainson, Arnold Ip, and Rohit Sriram.
